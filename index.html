Quant Trading Dashboard Feb – May 2026

I wanted to answer a question that bothered me: given real historical data, which trading strategy actually holds up — and how should you allocate across stocks to balance return against risk? Most retail tools give you a chart. I wanted the math underneath it, so I built the whole thing.

finance
Portfolio Theory Moving Avg Strategies Monte Carlo Simulation Value at Risk Sharpe Ratio Efficient Frontier Stress Testing

Python FastAPI React NumPy Pandas Pytest GitHub Actions Render Vercel
// finance
Backtests 15+ MA strategy combinations on real data, auto-selects the best by Sharpe ratio, and runs 5,000+ Monte Carlo simulations to map the efficient frontier across multi-asset portfolios.
// engineering
Full-stack system — FastAPI backend, React frontend, modular six-layer architecture, automated test suite, and a CI/CD pipeline gating every push. Deployed live.
